<h1>E-Hentai Helper</h1>
</div>

适用于 Chrome / Edge 等 Chromium 内核浏览器的 E-Hentai 画廊下载扩展。

✨ 如果这个扩展对你有帮助，欢迎回来给本项目点个 Star 🤗

## 安装

以 Google Chrome 为例：

1. 前往 [Release 页面](../../releases) 下载压缩包并解压
2. 打开 `chrome://extensions`
3. 勾选 **开发者模式**
4. 点击 **加载已解压的扩展程序**，选择解压后的扩展目录
5. 打开一个 E-Hentai 画廊，点击扩展图标
6. ✨ 开始下载！

## 功能

+ 一键下载整个画廊，不消耗 credit / GP，无需登录
+ 自由选择需要下载的图片范围
+ 提取并保存画廊信息与标签列表
+ 可选择文件命名规则（按序号或原始文件名）
+ 可选下载时转换图片格式
+ 实时查看下载进度，失败图片可重试
+ **历史记录**：曾下载过的画廊可续传缺失页，或从历史重新下载，无需从零开始
+ **CBZ 导出**（设置 → 输出）：下载完成后打包为单个 `.cbz`（仅 CBZ，或与单张文件同时保存）
+ 下载过程中可关闭弹窗，任务在后台继续
